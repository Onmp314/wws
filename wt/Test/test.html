Normal font

The normal font defaults to "lucida" size 12 points. There are various styles:
  • normal
  • bold
  • underlined

    The styles can be combined, eg. into bold underlined.

    Typewriter font

    The typewriter font defaults to "courier" 12 points. There are various styles as for the normal font:
    • normal
    • bold
    • italic
    • underlined
